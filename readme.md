# Projet React Task

## Préalable 

Télécharger et extraire l'archive du projet

ou cloner ce dépôt 

```bash
git clone git@github.com:ChaimaBdh/project-react-task.git
```

## Installation des fichiers

Dans le répertoire de l'archive extraite, exécuter la commande suivante :

```bash
npm install
```

## Execution du projet 

```bash
npm run dev-server
```
